## How to use

1. Open any prompt file
2. Copy the prompt block
3. Paste into your LLM of choice
4. Fill in the `[INPUT]` section at the bottom

Each prompt is self-contained. No setup required.

---

## Design principles

These prompts are built around four constraints that make LLM output actually useful:

- **Zero-base** — strip prior context before analysis
- **Default: No Action** — the base recommendation is always to do nothing until conditions are met
- **Falsification required** — every conclusion must include its own kill condition
- **Cui Bono** — follow the incentives, not the narrative
